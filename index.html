A step by step walk through of your first HTML page

HTML is short for Hyper Text Markup Language. Basically it's the "code" behind every web page- even as this one. If you are just beginning to learn HTML, let us tell you that it's fairly a easy task. HTML, without styling, can't do anything more that setting a layout, drawing a table or creating frames - but it is handy as it helps in structuring the content, which is important when you sit down to add style to your HTML.

However simple it might seem, it is a mighty useful tool, when it comes to full fledged web development. Various tool easily eliminate the HTML coding from your work process - but if you want to be in full control of your web page, you will need to have command over HTML.

Through this article, we aim to give you the basic essential HTML building blocks that will help you get and and running. Reading this you will be able to understand and HTML source code and even modify it for your own good!

Step one - TAGS

Tags are what you will see most when you'll look to an HTML source code. A tag can be seen as a wrapper to any item on any HTML document. Tags tell what magic is to be done to the content enclosed within them.

Let's look at two type of tags

  1. <tag-example-one>I need a closing tag <tag-example-one>
  2. <tag-example-two/> I don't need a closing tag

In the first example, the sentence is wrapped by two tags, the first one is called the opening tag and the seconnd one is called the closing tag. Everything in between is affected by the properties of the tag. Very commonly used examples of these tags are <html>, <body> etc.

The second example tag is of the loner tag- as in they dont need a closing tag to funtion. Although its not important these types of tags are often written as <tag/> to make the debugging of the code easier.
Common examples of these type of tags are <hr/> used for horizontal line.

Step two - HTML, HEAD and BODY: The three pillers of your document

These tags are essential for any HTML document. They parcel out the significant parts of your HTML code.

  • <BODY></BODY is placed below your <HEAD> ta, and everything that you wnat to be displayedon your screen comes under this tag. Text, images, linkks, and pretty much anything your browser live inside this tag.
  • <HTML></HTML> wraps your entire code. Everything else in your HTML document needs to come inside these tags
  • <HEAD></HEAD> includes things like title, style, and scrips. Head is usually present at the top (hah!), just inside the </HTML> tag.

Step three - A few tags that'll make your page pretty

Now that you know how to set up the skeleton of your document, let's proceed with the things that will go inside your <BODY> tag and do some magic!

Some basic text formatting tags:

  • <b> </b makes your text look bold
  • <i></i> makes you write in cursive
  • <u></u> underlines what you just wrote
For, example this piece of code

<html>
<head> </head>
<body>

<i> I am italics! </i> <br/>
<b> I am bold! </b> <;br/>
<u> And me, well, I'm underlined! </u> <br/>

</body>
</html>

Should prouduce something like this on your browser: Don't fret too much about the <br/>. It's just for breaking the line so that you can start from the next line. Enter key does little when it comes to changing lines in your HTML document.

Tags to help you structure your content:
  • <br/> breaks the line, making you continue to the next line.
  • <p> stands for paragraph. it divides your content into paragraphs.

Note: You need to use these tags as space and enter keys do very little when it comes to formatting content inside an HTML document.


Heading Tags:

HTML provides you with six tags, from <H1><H1> to <H6><H6> to help you create different sized headers quickly.

Inserting an Image

All that's good, but what fun without images on the webpage? Don't worry, <IMG/> to the rescue! The image tag has a mandatory attribute called "source". Basically, it tells the browser where it should look for the image. The syntax goes something like:

<img src="path_to_your_image"/>

Furthermore, it also has attributes like height and width that let you specify the height and width you want your image to take.

Lists:

HTML has two types of lists- ordered and unordered. Each item of your list has to be inclosed in a tag. The syntax for creating a list is fairly simple.

Suppose you want to create a list like:

  • Item 1
  • Item 2
  • Item 3

The following code will easily do the job for you:

<ul>
<li> Item 1 </li>
<li> Item 2 </li>
<li> Item 3 </li>
</ul>

This, by the way, was an example of an unordered list. For an ordered list, all you need to do is replace <ul> with <ol> and </ul> with </ol>.
